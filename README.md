# Athwela
### Installation

Athwela requires [Node.js](https://nodejs.org/) v4+ and [Angular CLI](https://cli.angular.io/) v8+ to run.

### Setup
Install the dependencies and start the server.

```sh
$ cd athwela-server
$ npm install -d
$ npm install -g nodemon
$ nodemon start
```

Install the dependencies start Angular client.

```sh
$ cd athwela-client
$ npm install -d
$ ng serve --open
```
